# 快速搭建我的世界地图


## Unity2017+Uniblock Voxel Terrain 1.4.1版本


通过插件更快速搭建游戏场景,自己写了一些主要逻辑.人物通过引入Unity内置的第一人称视角,自己编写了简单的日夜系统,点击销毁创建，
场景的最终搭建，方块的选择以及一些制作demo时的优化问题.优化及以后的一些完善方面我会放在Issues里面讲.
